Keliling Segitiga

Rumus Keliling Segitiga, yaitu :

K = a + b + c

Dimana :

K = keliling

a = sisi1

b = sisi2

c = sisi3

Hitung Keliling Segitiga

Luas Segitiga

Rumus Luas Segitiga, yaitu :

L = 1/2 * c * h

Dimana :

L = luas

c = panjang alas

h = tinggi

Keliling Jajargenjang

Rumus Keliling Jajargenjang, yaitu :

K = 2 * sisi1 * sisi2

Dimana :

K = keliling

AB & CD = sisi1

BC & AD = sisi2

Hitung Keliling Jajargenjang

Luas Jajargenjang

Rumus Luas Jajargenjang, yaitu :

L = a * t

Dimana :

L = luas

a = panjang alas

t = tinggi
